1) whereis 
Допустим, вы нашли нужную программу, но хотите попасть в ее директорию. Узнать директорию программы поможет whereis, показывающая полный путь к исполняемому файлу программы.

Example:
Для того чтобы найти все файлы, содержащие слово Mail, введите:
whereis Mail
Mail: /usr/bin/Mail /usr/lib/Mail.rc

2) history 
Отображает все недавно введённые вами команды.

Example:
history
1 mc
2 nano /etc/apt/sources.list
3 apt-get update
…

3) cp 
Создаёт копию нужного файла в папке терминала: cp путь_к_файлу. Также вы можете указать назначение cp путь_к_файлу путь_для_копии.

Example:
Для того чтобы скопировать файл из текущего каталога в другой каталог, введите:
cp jones /home/nick/clients

4) df
Показывает статистику по дисковому пространству: сколько места занято и сколько осталось.

Example:
df -h

5) mv
Для перемещения и переименования файлов и каталогов применяется команда mv. 
 
Example:
Для переименования файла и перемещения его в другой каталог введите:
mv intro manual/chap1

Для переименования файла введите:
mv appendix apndx.a

6) pwd
Укажет полный путь до директории, в которой вы находитесь. Если вы перепрыгнули на рабочий стол с помощью cd ~/Desktop/, то pwd покажет что-то в духе /home/Polzovatel/Desktop.

7) find
Позволяет задействовать одноименную утилиту, предназначенную для поиска файлов в иерархии директорий файловой системы. 

Example:
Например, для поиска файлов myClass.h, MyClass.h и так далее следует использовать команду:
find -iname myClass.h

8) grep
Команда grep предназначена для поиска строк, соответствующих шаблону, заданному в параметре Шаблон. Каждая найденная строка записывается в стандартный поток вывода.

Example:
Для просмотра всех строк вывода команды ls, которые содержат слово staff, введите:
ls -l | grep staff

9) !! 
Повторяет последнюю команду.

10) free
Показывает объём свободной и уже используемой памяти (альтернативная команда — free -g).

Example:
Вывод с использованием значений в мегабайтах:
free --mega
